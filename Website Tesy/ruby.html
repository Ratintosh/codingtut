Introduction to Ruby

Scroll down to learn how to make your first Ruby program!

Let's start off with a simple program, shall we?

We'll start off with ruby. Ruby is a coding language which runs inside of terminal. (Command Prompt for Windows users.)


Let's create our first ruby program!

First, you want to create a new file on your Desktop. Name this file "helloworld.rb"

Next, you want to right-click the file and select "Open With". Navigate over to "Notes" or any coding software you have installed.

Type the following code in:

puts "Hello World!"

Now open Terminal (Or Command Prompt) and type in the following commands:

cd Desktop
ruby helloworld.rb

If you were following the steps properly, the terminal would say "Hello World!"

Puts is a command used to display text in terminal.


Now, let's get to slightly more advanced programming!

Make a new line and type the following code in:

answer = gets.Chomp
if answer == 'Hello!' then
puts "Hello there!"
end

Okay, go back to terminal and type 'ruby helloworld.rb'

Now, there won't be any text shown on screen. That's normal. Try typing in the word 'Hello!'


See that? The terminal responded!

gets.Chomp is a command which awaits a response from the user.

if answer == 'Hello!' is a command stating if the user said a specifc word. NOTE THAT THIS IS CASE SENSITIVE!

But what if you said bye... The program would exit, right? Well, let's change that. Modify the original code to make it look like this:

answer = gets.Chomp
if answer == 'Hello!' then
puts "Hello there!"
elsif answer == 'Bye!' then
puts "Leaving so soon?"
end

That's really all there is to know about ruby. Sure, there's a lot more than just if then else, but this should get you started in making you an excellent ruby programmer!
